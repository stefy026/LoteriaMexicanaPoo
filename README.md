Lotería Mexicana - Windows Forms

Este proyecto es un juego de Lotería Mexicana desarrollado en C# con Windows Forms.  
La aplicación permite jugar una partida de lotería de forma visual, mostrando las cartas cantadas, los cartones del jugador, historial, sonido de cartas y funciones de red local para conectar jugadores.

Objetivo del proyecto

El objetivo principal es crear una aplicación interactiva que simule el juego tradicional de la lotería mexicana, aplicando conceptos de programación orientada a objetos, manejo de formularios, clases, eventos, listas, arreglos, validaciones y comunicación en red local.
